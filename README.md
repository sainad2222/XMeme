# [XMeme](https://my-xmeme.netlify.app/)

## what is Xmeme?
XMeme is a online meme submission platform made during crio.do CWOD 2021 extenship

## Technologies used:
### Frontend
 - React
 - Semantic-ui-react

### Backend
  - NodeJS
  - ExpressJS
  - MongooseJS

### Database
  - MongoDB Atlas
