# [XMeme](https://my-xmeme.netlify.app/)

### what is Xmeme?
XMeme is a online meme submission platform made during crio.do CWOD 2021 extenship

### Technologies used:
Database -> Mongodb

Frontend -> React

Backend  -> node+expressjs

Containers -> Docker

Styling  -> Semantic-UI-React
